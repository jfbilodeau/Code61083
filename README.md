# Demo Web App
## An ASP.NET demo app

This application is meant to demonstrate how to work with automation in _Azure DevOps_ and **GitHub**.

To push to GitHub, execute the following:

```bash
git init
git add -A *
git commit -m 'Initial commit'
git push
```
